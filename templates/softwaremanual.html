HOW TO USE & SETUP THE OPERATING SYSTEM

Below is a diagram of the probe head

Setup

  1. Create your first profile in the 'Input Parameters' section by giving it a name and clicking the submit button. If no name is provided, it will be given the default name: "Profile x - DD/MM/YYYY at HR/MIN/SEC".
  2. Once created, the most recently added profile will appear in the 'Current Profile' dropdown on the main page.

Inputting Parameters

  1. In the 'Input Parameters' section, enter all the selected values for the fan motor, linear actuator, and rotary motor.
  2. Switch back to the main page to view all the entered values in the corresponding sections.

Calibration

  1. Click the 'Calibrate' button to calibrate the load cells. The calibration offsets will be displayed.

Start Experiment

  1. Once satisfied with the setup and parameter inputs, click the 'Start' button to begin the experiment.
  2. The relevant output parameters and charts will be displayed below.

Stop Experiment

  1. To stop collecting data at any time, click the 'Stop' button.

Exporting Data

  1. If you wish to export all your values to a CSV file, click the 'Export CSV' button.

Test Profiles

  1. In the 'Test Profiles' section, you can create and manage test profiles for different scenarios.

System Limits

  1. Ensure that all input parameters are within the specified system limits to maintain safety and prevent damage.
