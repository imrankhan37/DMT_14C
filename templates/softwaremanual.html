DMT 14C: Aerodyanmic probe and data acquisition software - How to use

Below is a step-by-step guide on how to use the software

  1. In the 'Input Parameters' subpage, create your first profile by giving it a name and click the submit button. If you choose to not give it any name, it will be given the default name: "Profile x - DD/MM/YYYY at HR/MIN/SEC".
  2. If you go back to the main page, the profile you have most recently created should show up under the 'Current Profile' dropdown.
  3. Go back to the 'Input Parameters' subpage and enter all your selected values for the fan motor, linear actuator, and rotary motor.
  4. Go back to the main page again, you should see all your entered values.
  5. Click the start button once you are happy with everything.
  6. All the relevant output parameters and charts will show up below.
  7. Click the stop button at any time when you wish to stop collecting data.
  8. Should you wish to export all your values to a CSV file, click the 'Export CSV' button.
